137 // There is no way to actually detect the libstdc++ version; __GLIBCXX__
138 // is always set to the date of libstdc++ being packaged, not the release
139 // day or version. This means that we can't detect the libstdc++ version,
140 // except when compiling with GCC.
141 //
142 // Therefore, for the best approximation of is_trivially_copyable, we need to
143 // handle three distinct cases:
144 // 1) GCC above 5, or another C++11 compiler not using libstdc++: use the
145 // standard trait directly.
146 // 2) A C++11 compiler using libstdc++ that provides the intrinsic: use the
147 // intrinsic.
148 // 3) Any other case (essentially: compiling without C++11): has_trivial_assign.
