__host__ __device__ pointer< T > thrust::system::cuda::malloc ( std::size_t  n)
inline

Allocates an area of memory available to Thrust's cuda system.

Parameters
nNumber of bytes to allocate.
Returns
A cuda::pointer<void> pointing to the beginning of the newly allocated memory. A null cuda::pointer<void> is returned if an error occurs.
Note
The cuda::pointer<void> returned by this function must be deallocated with cuda::free.
