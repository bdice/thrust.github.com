Allocates an area of memory available to Thrust's tbb system.

Parameters
nNumber of bytes to allocate.
Returns
A tbb::pointer<void> pointing to the beginning of the newly allocated memory. A null tbb::pointer<void> is returned if an error occurs.
Note
The tbb::pointer<void> returned by this function must be deallocated with tbb::free.
