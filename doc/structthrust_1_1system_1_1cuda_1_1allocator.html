Detailed Description

template<typename T>
struct thrust::system::cuda::allocator< T >

cuda::allocator is the default allocator used by the cuda system's containers such as cuda::vector if no user-specified allocator is provided. cuda::allocator allocates (deallocates) storage with cuda::malloc (cuda::free).
