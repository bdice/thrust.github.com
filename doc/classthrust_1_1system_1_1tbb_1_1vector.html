Detailed Description

template<typename T, typename Allocator = allocator<T>>
class thrust::system::tbb::vector< T, Allocator >

tbb::vector is a container that supports random access to elements, constant time removal of elements at the end, and linear time insertion and removal of elements at the beginning or in the middle. The number of elements in a tbb::vector may vary dynamically; memory management is automatic. The elements contained in a tbb::vector reside in memory available to the tbb system.

Template Parameters
TThe element type of the tbb::vector.
AllocatorThe allocator type of the tbb::vector. Defaults to tbb::allocator.
